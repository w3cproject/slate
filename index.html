    Overview

    BASE URL: https://api.zuzan.pro/v1

    Welcome to the Zuzan REST API comrad!

    This is base API for all zuzan resources based on python.

    We have language bindings in Shell, Python! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Python requirements

    Version: >=3.5.3

    JWT library: Flask JWT

    API Framework: Flask-RESTful

    Server: Gunicorn

    DataBase: MySQL

    Authentication

    Zuzan uses JSON Web Token Authentication to allow access to the API.

    Before send any request, you must be authorized with your personal email and password

    Zuzan expects for the token to be included in all API requests to the server in a header that looks like the following:

    Authorization: Bearer ${ACCESS_TOKEN}

    You must replace ${ACCESS_TOKEN} with your `token`.

    Login

    To authorize, use this code:

    # Make sure to replace `test` with your login and password.
curl -X POST "https://api.zuzan.pro/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d '{"email":"test","password":"test"}'

    import zuzanApi

api = zuzanApi.authorize('token')

    The above command returns JSON structured like this:

    {
"access_token": "${ACCESS_TOKEN}",
"refresh_token": "${REFRESH_TOKEN}",
"user": {}
}

    HTTP Request

    POST /auth/login

    Query Parameters

    Parameter Type Required Description
    email string yes User email
    password string yes User password

    Logout

    To logout, use this code:

    # Make sure to replace `${ACCESS_TOKEN}` with your user `access_token`.
curl -X DELETE "https://api.zuzan.pro/v1/auth/logout" \
    -H "Authorization: Bearer ${ACCESS_TOKEN}"

    import zuzanApi

api = zuzanApi.authorize('token')

    The above command returns JSON structured like this:

    {
"message": "Successfully logged out",
"status": "success"
}

    HTTP Request

    DELETE /auth/logout

    Description

    Deleting information about user auth from cookies

    Errors

    This error section is stored in a separate file in includes/_errors.md. Slate allows you to optionally separate out your docs into many files...just save them to the includes folder and add them to the top of your index.md's frontmatter. Files are included in the order listed.

    The ZuzanApi API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Bad credentials, please verify that your username/password are correctly set.
    403 Forbidden -- Your token is invalid, please login again to get a new one.
    403 Forbidden -- Missing token.
    404 Not Found -- The specified object could not be found.
    405 Method Not Allowed -- You tried to access a object with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The object requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many requests! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
